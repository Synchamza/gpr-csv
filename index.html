How to use

  1. Upload a .gpr file exported from GenePix Pro
  2. Select the pad number (sample position 1-16)
  3. Enter an optional test reference for the patient
  4. Click Process to download the result CSV
Note: A single GPR file can contain up to 16 pads (samples). Process each pad separately to get individual reports.
