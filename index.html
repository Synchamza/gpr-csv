How to use

  1. Upload a .gpr file exported from GenePix Pro

  3. Enter optional test references per pad
  4. Click Process & Download ZIP to get all CSVs
Note: A single GPR file can contain up to 16 pads. Select multiple pads to download them all in one ZIP file.
